│                        🤖 AI INTEGRATION STRUCTURE                           │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  OpenAI Service Layer (lib/services/openai_service.dart)                   │
│  ┌────────────────────────────────────────────────────────────────┐        │
│  │                                                                 │        │
│  │  📝 getAICoachResponse(message)                                │        │
│  │     ↓                                                           │        │
│  │     Creates fitness coaching prompt                            │        │
│  │     ↓                                                           │        │
│  │     Calls ApiPlaceholders.getAIResponse(prompt)                │        │
│  │     ↓                                                           │        │
│  │     Returns AI-generated fitness advice                        │        │
│  │                                                                 │        │
│  │  💪 getWorkoutSuggestion(level, goal, minutes)                 │        │
│  │     ↓                                                           │        │
│  │     Creates workout generation prompt                          │        │
│  │     ↓                                                           │        │
│  │     Calls ApiPlaceholders.getAIResponse(prompt)                │        │
│  │     ↓                                                           │        │
│  │     Returns structured workout plan                            │        │
│  │                                                                 │        │
│  └────────────────────────────────────────────────────────────────┘        │
│                                                                              │
│  💡 To Enable Real AI:                                                      │
│  1. Get OpenAI API key from https://platform.openai.com                    │
│  2. Update lib/utils/api_placeholders.dart                                 │
│  3. Replace placeholder with actual OpenAI API call                        │
│  4. See TESTING.md for complete integration code                           │
